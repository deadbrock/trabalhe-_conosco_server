# Backend - Trabalhe Conosco

## Requisitos
- Node.js 18+
- PostgreSQL 13+
- WPPConnect para WhatsApp

## Setup
1. Copie `.env.example` para `.env` e ajuste as variáveis.
2. Instale dependências:
   ```bash
   npm install
   ```
3. Rode migrações:
   ```bash
   npm run migrate
   ```
4. Suba o servidor:
   ```bash
   npm run dev
   ```
